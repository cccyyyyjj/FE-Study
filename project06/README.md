# 유동적인 dt + dd 형태 만들기!
[제목] : [내용]이 인라인 형태로 붙는 형태로, 제품설명이나 이런데에 많이 쓰이는 UI입니다.
마크업은 아래 구조대로 하시고 css 만 하시면 되요!
```html
<dl>
<dt>제목</dt>
<dd>내용입니다.</dd>
</dl>
```
