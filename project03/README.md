# #03 과제
## 해상도에 따라 컬럼이 늘어나는 리스트의 구분선 유지하기~

## 스펙
- 구분선은 각 리스트 사이에 들어가며 첫 상단과 마지막 하단에는 들어가지 않는다.
- 리스트 형태는 아래 그림처럼 모바일에서 1단, 태블릿이상에서는 2단으로 설정!
- 해상도 분기점은 640을 기준으로 이하는 모바일 / 그 이상은 태블릿으로 설정!

- 내용2
- -----(구분선)------
- 내용3
- -----(구분선)------
- 내용4
- -----(구분선)------
- 내용5

- 내용6
